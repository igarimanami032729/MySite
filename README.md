# MySite
 이번 업데이트에서는 페이지 탐색과 섹션 전환을 자연스럽고 직관적으로 만들기 위한 다양한 기능을 추가하였습니다. 

## 주요 기능

### 1. 상단 메뉴 클릭 시 부드러운 스크롤 (Smooth Scroll)
- 기능 설명: 사용자가 상단 메뉴에서 특정 항목을 클릭하면, 해당 섹션으로 스크롤이 부드럽게 이동합니다. 페이지 이동이 깔끔하고 자연스럽게 이루어지며, 사용자가 빠르게 원하는 내용을 확인할 수 있도록 도와줍니다.
- 기술 구현: `HTML`과 `CSS`의 기본 마크업 구조 위에 `JavaScript`로 스크롤 이벤트를 처리하고, `scrollIntoView()` 메서드를 활용하여 애니메이션처럼 부드럽게 이동합니다.
- 사용자 경험: 사용자가 상단 메뉴에서 클릭하는 순간, 스크롤이 멈추지 않고 매끄럽게 섹션을 따라가므로 페이지 내비게이션이 직관적이고 편리해집니다.

### 2. 스크롤 시 섹션 애니메이션 (Fade-in/Slide-in)

기능 설명: 페이지를 스크롤할 때 특정 섹션이 화면에 들어올 때 애니메이션 효과가 적용됩니다. 섹션은 부드럽게 나타나거나(slide-in), 혹은 점점 투명해지며(fade-in) 나타나게 됩니다.
기술 구현: `IntersectionObserver` API를 사용하여 특정 섹션이 화면에 보이게 될 때, CSS `transition`과 `transform` 속성을 활용해 애니메이션을 적용합니다.
사용자 경험: 사용자가 페이지를 내려갈 때, 애니메이션 효과로 콘텐츠가 동적으로 등장하면서 페이지에 생동감을 더합니다. 이를 통해 사용자는 페이지 내에서 더 많은 정보를 직관적으로 얻을 수 있습니다.

### 3. TOP 버튼 클릭 시 페이지 맨 위로 이동 (Back to Top)

기능 설명: 사용자가 페이지 하단에 위치한 **TOP 버튼**을 클릭하면 페이지가 맨 위로 부드럽게 스크롤됩니다. 이 기능은 긴 페이지에서 사용자가 편리하게 페이지 최상단으로 돌아갈 수 있도록 도와줍니다.
기술 구현: `window.scrollTo()` 메서드를 사용하여, 버튼 클릭 시 페이지 상단으로 부드럽게 이동할 수 있도록 설정합니다.
사용자 경험: 페이지를 스크롤한 후 하단에 위치한 버튼을 클릭하면, 짧은 시간 안에 최상단으로 부드럽게 이동하며 빠르게 페이지 탐색을 재개할 수 있습니다.

## 기술 스택

HTML5: 웹 페이지의 구조를 작성하는 데 사용됩니다.
CSS3: 웹 페이지의 스타일을 정의하며, 애니메이션과 트랜지션 효과를 구현합니다.
JavaScript: 페이지 동작, 스크롤 및 애니메이션 효과를 구현하는 데 사용됩니다.
IntersectionObserver API: 특정 요소가 화면에 나타날 때 이벤트를 트리거하여 애니메이션을 적용하는 데 사용됩니다.
CSS Transitions: 요소의 변화를 부드럽게 애니메이션 처리합니다.
